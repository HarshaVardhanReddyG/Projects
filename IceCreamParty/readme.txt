This program takes input from a file named "inp.txt" which contains 7 integers k,m,n,a,b,l,u respectively.

On Linux,
open the the terminal and navigate to the directory containing "Assgn4-Src-CS21BTECH11017.cpp" and "inp-params.txt"

Then compile the c++ code for tas and run using
$ g++ 1.cpp
$ ./a.out < inp.txt


Running above commands will produce an output file named "output.txt" in the same directory.
