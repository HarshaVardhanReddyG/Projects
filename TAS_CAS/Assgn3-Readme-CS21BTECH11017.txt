This program takes input from a file named "inp-params.txt" which contains four integers n, k, lambda1, lambda2 respectively.

On Linux,
open the the terminal and navigate to the directory containing "Assgn3-Src-tas-CS21BTECH11017.cpp", "Assgn3-Src-cas-CS21BTECH11017.cpp", "Assgn3-Src-cas-bounded-CS21BTECH11017.cpp" and "inp-params.txt"

Then compile the c++ code for tas and run using
$ g++ Assgn3-Src-tas-CS21BTECH11017.cpp
$ ./a.out < inp-params.txt

Then compile the c++ code for cas and run using
$ g++ Assgn3-Src-cas-CS21BTECH11017.cpp
$ ./a.out < inp-params.txt

Then compile the c++ code for cas-bounded and run using
$ g++ Assgn3-Src-cas-bounded-CS21BTECH11017.cpp
$ ./a.out < inp-params.txt

Running above commands will produce output files "tas_me_output.txt", "cas_me_output.txt", "cas_bounded_me_output.txt" respectively in the same directory.
